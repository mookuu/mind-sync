# UX 交互规范 — mind-sync

> 本文档描述页面与组件的交互行为，**区别于 PRD 的功能描述**。
> PRD 定义 WHAT，本文档定义 HOW。

---

# 第一部分：全局规范

## 1. 布局与导航

### 1.1 侧边栏基本行为

| 规则 | 描述 |
|------|------|
| 唯一活性 | 同一时刻只有一个父级菜单 / 子级菜单处于选中状态 |
| 切换展开 | 切换到有子级的父级菜单时，展开该父级 + 选中第一个子级 |
| 切换折叠 | 从有子级的父级 A 切换到父级 B 时，A 自动折叠，B 展开 |
| 账户固定 | 「账户」始终在侧边栏底部，不参与菜单切换 |

### 1.2 文档树（侧边栏子级）

| 场景 | 行为 |
|------|------|
| 初次打开 | 无子级被选中、所有子级折叠、文档区显示「选择左侧文件开始阅读」 |
| 非初次（同次登录） | 保留上次展开的树结构 + 选中的子级/文档 |
| 跨登录 | 同上，状态持久化到 localStorage |
| 从搜索跳转 | 自动展开并定位到匹配文档 |

### 1.3 顶栏

| 元素 | 行为 |
|------|------|
| 页面标题 | 显示当前路由 meta.title |
| 角色标签 | admin=蓝色「管理员」/ member=灰色「成员」 |
| 通知铃铛 | 未读时显示红点 + 数字；点击展开通知列表 |

---

## 2. 页面状态保持

| 场景 | 行为 |
|------|------|
| 页面切换（切走再切回） | 保留前回显示内容 + 状态（搜索词、展开的树、滚动位置等） |
| 登录/登出 | 保留（登出不清除，登录后恢复） |
| F5 刷新 | 保留 |
| Ctrl+F5 强制刷新 | 保留 |
| 服务器重启 | 不保留（可选优化，当前无需求） |

## 3. 通用组件

### 3.1 弹窗（Modal）

| 规则 | 描述 |
|------|------|
| 实现方式 | 全部用 Vue 组件式（禁止 alert/confirm/prompt） |
| 关闭方式 | 点击遮罩层 / ESC 键 |
| 位置 | 居中 |
| 覆盖 | 显示期间背景页面不可操作 |

### 3.2 Toast 通知

| 规则 | 描述 |
|------|------|
| 位置 | 顶部居中 |
| 持续时间 | 成功 = 3s / 错误 = 5s / 警告 = 5s |
| 可关闭 | 错误/警告 toast 可手动关闭 |
| 不堆积 | 同一类型 toast 替换前一个 |

### 3.3 通知条（NotifyBar）

| 场景 | 触发 |
|------|------|
| 源被管理员设为共享 | 源所有者收到通知 |
| 源被管理员删除 | 源所有者收到通知 |
| 通知已读 | 点击通知跳转目标页 + 自动标记已读 |
| 未读数量 | 顶栏铃铛图标显示红点 + 数字 |

## 4. 状态反馈

### 4.1 Loading

| 场景 | 行为 |
|------|------|
| 页面数据加载中 | 居中 spinner + 「加载中…」文字 |
| 按钮操作执行中 | 按钮变为 loading + disabled |
| 后台任务（sync/rebuild） | 顶栏通知条显示进度 |
| 首次加载（冷启动） | Skeleton 占位符（⚡ 当前未实现，建议后续添加） |

### 4.2 Error

| 场景 | 行为 |
|------|------|
| API 401 | 自动跳转登录页 |
| API 403 | 红色 toast「权限不足」；管理页面跳转回文档库 |
| API 404 | 红色 toast「资源未找到」 |
| API 429 | 黄色 toast「操作过于频繁，请稍后再试」+ Retry-After 倒计时 |
| API 500 | 红色 toast「服务器内部错误」+ 建议稍后重试 |
| 网络断开 | 红色 toast「网络连接失败」；不跳转，保留当前页面状态 |
| 数据加载失败 | 页面内显示错误信息 + 「重试」按钮 |

### 4.3 Empty

| 场景 | 行为 |
|------|------|
| 搜索无结果 | 显示「未找到匹配内容」友好提示 |
| 文档库无文档 | 显示「选择左侧文件开始阅读」 |
| 图谱无数据 | 显示「暂无数据，请先创建 Wiki 页面」 |
| 用户列表为空 | 显示「暂无用户」 |

---

## 5. 权限控制

| 场景 | 行为 |
|------|------|
| 非管理员访问管理页面 | 跳转到文档库 + 黄底提示「页面『xxx』仅管理员可访问」 |
| 管理员专属菜单项 | 完全不显示（非管理员不可见） |
| 管理员专属操作按钮 | 非管理员视图中隐藏（同步按钮除外——member 可同步自己的源） |
| 写入操作（Wiki/设置等） | 非管理员按钮 disabled 或隐藏；API 层返回 403 |

---

## 6. 响应式行为

| 断点 | 布局 |
|------|------|
| >1024px | 侧边栏固定 240px + 内容区自适应 |
| 768-1024px | 侧边栏可折叠（hamburger 按钮） |
| <768px | 🔒 暂不适配，留待后续有移动端使用需求时再实施 |

# 第二部分：用户功能

## 7. 登录页

### 7.1 表单

| 操作 | 行为 |
|------|------|
| 输入用户名 + 密码 → 点击登录 | 提交认证；成功跳转文档库，失败显示错误提示 |
| 回车 | 同点击登录 |
| 空用户名或空密码 → 登录 | 按钮 disabled，不可提交 |
| 勾选「记住我」 | 会话有效期延长至 30 天（默认 7 天） |

### 7.2 状态

| 状态 | 行为 |
|------|------|
| 登录中 | 按钮显示 loading 状态，禁止重复提交 |
| 登录失败（401） | 红色提示「用户名或密码错误」 |
| 登录失败（423） | 红色提示「账户已锁定，请 5 分钟后再试」 |
| 登录失败（429） | 提示「登录尝试过多，请稍后再试」+ 倒计时 |
| 登录失败（网络错误） | 提示「网络连接失败，请检查服务是否启动」 |
| Session 过期 | 自动跳转到登录页，显示「会话已过期，请重新登录」 |

### 7.3 外观

| 元素 | 描述 |
|------|------|
| 标题 | 「mind-sync」居中显示 |
| 输入框 | 用户名在上，密码在下 |
| 按钮 | 蓝色主按钮「登录」 |
| 版权 | 底部居中显示版本信息 |

---

## 8. 文档库

### 8.1 文档渲染

| 文档类型 | 显示行为 |
|----------|----------|
| Markdown (.md) | 正确渲染：表格、图片、链接、代码块 |
| 代码 (.py/.java/…) | 语法高亮（语言识别 + 颜色标记） |

### 8.2 搜索高亮联动

| 场景 | 行为 |
|------|------|
| 当前文档在搜索结果列表中 | 所有搜索关键字高亮 |
| 右侧「重置搜索结果」按钮 | 可见，点击后清除所有高亮、恢复键盘原始功能、保持文档打开 |
| 从搜索跳转 | 自动滚动使第一个高亮在屏幕居中 |

### 8.3 键盘操作

| 场景 | ↑↓ 行为 |
|------|----------|
| 从搜索跳转的文档 | 切换上一个/下一个高亮关键字 |
| 手动打开文档（在搜索结果中） | 同上 |
| 手动打开文档（不在搜索结果中） | 上下滚动页面 |
| 从其他页面切回文档库 | 根据前回显示的文档行为恢复 |

### 8.4 权限拦截

| 场景 | 行为 |
|------|------|
| 文档在私有源中且无权限 | 显示「此文档属于私有知识库，你无权访问」提示 |
| Wiki 页面属于其他用户 | 同上（403） |

---

## 9. 搜索页

### 9.1 输入与触发

| 操作 | 行为 |
|------|------|
| 输入关键字 → 点击搜索 | 执行搜索 |
| 输入关键字 → 回车 | 同点击搜索 |
| 空输入 → 点击搜索 | 清空结果，显示「未找到匹配内容」 |
| 点击「重置」按钮 | 有结果时显示，点击后清空结果回到初始页 |

### 9.2 筛选与排序

| 操作 | 行为 |
|------|------|
| 切换排序方式（相关度/最近修改） | 即时重新排序，无需重新搜索 |
| 切换分类（全部/原始素材/学习摘要/问答沉淀） | 即时过滤，无需重新搜索 |

### 9.3 结果展示

| 场景 | 行为 |
|------|------|
| 0 件 | 显示「未找到匹配内容」 |
| 多件 | 分页（每页 5 件），标明文档来源 + 角色可见标签 |
| 关键字高亮 | 搜索结果摘要中高亮匹配关键字 |
| 私有源结果 | 标注 🔒 图标 |

### 9.4 结果 → 文档跳转

| 操作 | 行为 |
|------|------|
| 单击搜索结果 | 跳转至文档库并打开文档（参见 8.2） |

## 10. 问答页

### 10.1 输入

| 操作 | 行为 |
|------|------|
| 输入问题 → 提交 | 发送到 LLM，基于 FTS 检索结果回答 |
| 空输入 | 提交按钮 disabled |
| 无 LLM 配置 | 降级为检索摘要模式（仅返回相关文档片段） |

### 10.2 回答展示

| 元素 | 行为 |
|------|------|
| 回答内容 | Markdown 渲染 |
| 证据引用 | 每条引用显示置信度标签 + 来源路径；点击跳转到文档 |
| 置信度标签 | EXTRACTED=绿色 / INFERRED=蓝色 / AMBIGUOUS=黄色 / UNVERIFIED=灰色 |
| 保存到 Wiki | admin 可见按钮：保存当前问答到 `wiki/queries/` |

### 10.3 加载状态

| 状态 | 行为 |
|------|------|
| 检索中 | 显示「正在检索相关知识…」 |
| LLM 生成中（当前） | 显示「正在生成回答…」，等待完整响应后一次性展示 |
| LLM 生成中（流式） | ⚠️ 待实施：token-by-token 逐字显示，用户可看到答案实时生成 |
| 流式 + 推理过程 | ⚠️ 待实施：DeepSeek-R1 reasoning_content 折叠显示在答案前 |
| 超时/失败 | 显示错误信息 + 重试按钮 |

### 11.2 交互

| 缩放 | 鼠标滚轮缩放画布 |
| 孤立节点 | 特殊样式标识（灰色/虚线边框） |

## 12. 账户页

### 12.1 信息展示

| 元素 | 描述 |
|------|------|
| 用户信息卡片 | 显示用户名、显示名、角色标签（管理员/成员） |
| 角色标签样式 | admin=蓝色徽章，member=灰色徽章 |

### 12.2 修改密码

三个输入框：当前密码、新密码、确认新密码。

| 操作 | 行为 |
|------|------|
| 输入当前密码 + 新密码 + 确认新密码 → 提交 | 成功提示「密码已更新」；失败提示错误原因 |
| 新密码 < 4 字符 | 按钮 disabled + 输入框下方红色提示 |
| 确认新密码与新密码不一致 | 输入框下方红色提示「两次输入的密码不一致」；提交按钮 disabled |

### 12.3 会话管理

| 操作 | 行为 |
|------|------|
| 查看活跃会话 | 列表显示：IP、User-Agent、登录时间、最后活跃时间 |
| 吊销指定会话 | 点击删除按钮 → 确认弹窗 → 该会话立即失效 |
| 吊销当前会话 | 等同于登出 |

### 12.4 API Key 管理（admin 专属）

| 操作 | 行为 |
|------|------|
| 查看 Key 列表 | 显示 Key 前缀（`msk-xxxx…`）、标签、创建时间、绑定用户 |
| 生成新 Key | 输入标签 → 点击生成 → 显示完整 Key（仅此一次可见） |
| 复制 Key | 点击复制按钮 → 复制到剪贴板 + 「已复制」提示 |
| 删除 Key | 确认弹窗 → 该 Key 立即失效 |

# 第三部分：管理功能

## 13. 同步控制

### 13.1 同步操作

| 操作 | 行为 |
|------|------|
| 点击「增量同步」 | 后台执行 sync；按钮变为 loading + 显示「同步中…」 |
| 点击「全量重建」 | 确认弹窗 → 后台执行 rebuild |
| 同步进行中 | 两个按钮均 disabled；显示进度信息 |
| 同步完成 | 显示「成功」/「失败」状态 + 耗时；失败显示错误详情 |

### 13.2 同步范围

| 操作 | 行为 |
|------|------|
| 切换预设（全部/自定义/仅共享） | 自动保存到服务端，下次同步使用新范围 |
| 选择自定义源 | 勾选/取消勾选 → 自动保存 |
| member 用户 | 只显示自己的源 + 共享源，预设中「全部」=自己的全部源 |

### 13.3 自动同步

| 操作 | 行为 |
|------|------|
| 启用/禁用 | 切换开关 → 自动保存 |
| 下次同步时间 | 基于设置间隔计算，实时更新 |
| 上次同步状态 | 显示最近一次同步的成功/失败 + 时间 |

## 14. 素材管理

### 14.1 管理员视图

| 区域 | 显示内容 |
|------|----------|
| 全局知识库 | 所有预设源 + 自定义源列表，可添加/删除/编辑 |
| 我的知识库 | 管理员的私有源 |
| 成员知识库 | 按用户分组显示成员的私有源 + 共享状态 |

### 14.2 成员视图

| 区域 | 显示内容 |
|------|----------|
| 共享知识库 | 只读列表（管理员配置的源） |
| 我的知识库 | 可添加/删除自己的私有源 |

### 14.3 源操作

| 操作 | 行为 |
|------|------|
| 添加源 | 弹出路径输入框 + 类型选择；验证路径存在性 |
| 删除源 | 确认弹窗 → 删除源配置 + 索引数据 |
| 切换共享 | admin 可将成员源设为共享；切换后通知源所有者 |
| 路径不存在 | 源行显示 ⚠️ 警告图标 + 提示文字 |
| 目录浏览器 | 点击浏览按钮 → 文件系统树形选择器 |

## 15. 用户管理（admin 专属）

### 15.1 用户列表

| 元素 | 行为 |
|------|------|
| 排序 | admin 在前，按创建时间排序 |
| 角色列 | 下拉选择框：admin / member；切换即时生效 |
| 状态列 | normal=绿色 / locked=红色 / deleted=灰色 |
| 源数量 | 显示该用户拥有的私有源数量 |

### 15.2 创建用户

| 操作 | 行为 |
|------|------|
| 点击「创建用户」 | 弹出表单：用户名、显示名、密码、角色 |
| 提交 | 后端验证 → 成功关闭弹窗 + 刷新列表；失败显示错误 |
| 用户名重复 | 提示「用户已存在」 |
| 密码 < 4 字符 | 前端校验，提交按钮 disabled |

### 15.3 用户操作

| 操作 | 行为 |
|------|------|
| 修改角色 | 下拉框切换 → 确认弹窗 → 即时更新 DB |
| 重置密码 | 输入新密码 → 确认 → 成功提示 |
| 删除用户 | 确认弹窗（提示不可逆）→ 软删除，保留数据目录 |
| 不可删除自己 | 按钮 disabled + tooltip 提示 |

---

## 16. 仓库管理（Vault）

### 16.1 Git Vault

| 操作 | 行为 |
|------|------|
| 查看状态 | 显示当前分支、未提交变更、远程差异 |
| Push | 推送本地 Wiki 变更到远程 |
| Pull | 拉取远程变更（⚠️ 覆盖本地 Wiki） |
| Pull 前确认 | 弹窗警告「拉取将覆盖本地 Wiki 内容」 |
| 未配置 Vault | 显示「未配置 Git Vault」+ 配置引导链接 |
