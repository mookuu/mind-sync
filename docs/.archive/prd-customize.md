## 功能描述

### 共有功能

1. 菜单
   - 同一时刻只有一个菜单/子级菜单处于活性、选中状态
   - 切换菜单时，如果切换的菜单有子级菜单，则展开一级子菜单并选中第一个一级子菜单。如果该一级子菜单是页面，则显示该页面
     - 如切换到父级菜单 同步运维，则 同步运维 下的 同步控制 子菜单默认被选中，并显示同步控制页面
   - 切换菜单时，如果切换前选中的菜单有子级菜单，则将切换前的父级菜单折叠
     - 如从 同步运维 切换到 系统管理，则折叠父级菜单 同步运维
   - 特殊：文档库
     - 文档库初次打开
       - 默认无子级菜单被选中
       - 所有子级菜单被折叠
       - 显示文档库初始页面，提示 "选择左侧文件开始阅读"
     - 文档库非初次打开
       - 菜单栏保留上次的树状结构，并选中前回打开的子级菜单；如果前回打开了文档，则选中该文档
         - 如：文档库-学习摘要-Wiki摘要-SCHEMA.md

2. 页面保持
   - 概要：保留前回页面显示内容、功能。如果是初次显示，则显示相应的初始化页面
   - 前提：同一用户
   - 场景
     - 页面切换
       - 如 初次搜索页面 显示 初始化搜索页面
       - 关键字搜索后，再次从其他页面迁移到搜索页面，显示前回关键字搜索结果
     - 登入登出
     - F5重新加载
     - Ctrl+F5强制重新加载
   - 例外：服务器重启无需保持前回页面显示内容 -> TODO：服务器重启对页面保持影响调查，能否实现重启也能保持前回显示的页面，有无实现必要？

3. 弹窗
   - 所有弹窗均使用vue组件式弹窗
   - ESC按钮关闭弹窗
   - 所有弹窗居中表示
   - 弹窗均为覆盖式：弹窗显示后所有页面均无法操作。关闭弹窗后页面可操作

4. 权限
   - 菜单

       - 搜索
       - 文档库
       - 知识查询
       - 同步运维
         - 同步控制
         - 同步素材

         - 规则约束
         - 操作记录
       - Wiki图谱
       - 系统管理
         - 系统概览
         - 用户管理
     - 成员角色：下面菜单可见、可操作
       - 搜索
       - 文档库

       - 同步运维

     - 成员角色：菜单对应页面，各页面内权限详见同步运维功能

### 搜索

- 输入框
  - 输入关键字后，点击搜索按钮可搜索出相关文档
  - 当空输入并点击搜索按钮，清空页面显示无结果的友好提示
  - 按钮关闭
- 排序方式下拉框
  - 相关度：搜索结果按照相关度进行排序
  - 最近修改：搜索结果按照文档的最近修改时间进行排序
- 分类下拉框
  - 全部分类：搜索对象为所有文档
  - 原始素材：搜索对象仅为原始素材
  - 学习摘要：搜索对象仅为学习摘要
  - 问答沉淀：搜索对象仅为问答沉淀
- 搜索按钮
  - 使用"查询关键字+分类" 搜索文档，并按照设置的排序方式对结果进行排序
  - 输入关键字后，回车键可触发搜索动作
- 重置按钮
  - 搜索按钮右侧
  - 无搜索结果时非表示
  - 有搜索结果时表示，点击后清空搜索结果并显示搜索初始页面，并再次隐藏重置按钮
- 搜索结果
  - 0件：显示无结果的友好提示
  - 多件
    - 分页表示，每页5件
    - 切换排序方式、分类时，可自动过滤相关结果并排序，无需重新搜索
    - 搜索结果标明文档来源：原始素材、学习摘要、问答沉淀
    - 搜索结果高亮查询关键字
  - 单击搜索结果
    - 页面跳转至文档库页面并打开相应文档
      - 文档内高亮所有查询关键字
      - 调整页面使第一个高亮关键字所处行居中表示
      - 键盘上下箭头匹配到的上一个/下一个关键字
    - 菜单栏展开文档树并选中匹配文档
      - 如：文档库-学习摘要-Wiki摘要-SCHEMA.md（关键字匹配文档）

### 文档库

- 概要：树状形式显示原始素材、学习摘要、问答沉淀文档
- 功能：
  - 打开md文档、代码等并正确显示
    - md文档：能正确显示表、图片、链接、代码块等，实现常规md文档显示功能
    - 代码：识别常见编程语言并能正确表示代码，如语法、包、类等能以不同颜色标记
  - 打开的文档是搜索结果文档列表中的文档时，高亮显示所有搜索关键字
- 文档右侧按钮：重置搜索结果
  - 显示条件：当前打开文档在搜索结果文档列表中时
  - 动作
    - 清空搜索页面的搜索结果
    - 恢复键盘上下箭头原始功能
    - 取消所有文件的高亮显示
    - 保持当前文档打开、正常显示无高亮
- 键盘按钮功能更改：
  - 上下箭头：
    - 从搜索结果页面跳转至文档库页面时：切换匹配到的上一个/下一个关键字
    - 手动打开在搜索结果文档列表中的文档时：切换匹配到的上一个/下一个关键字
    - 手动打开不在搜索结果文档列表中的文档时：原始功能，即如果页面有滚动条的话则上下滑动页面
    - 从其他页面再次切换到文档库时：根据前回显示的文档决定上下键功能
      - 前回显示的文档为搜索结果文档列表中的文档时：切换匹配到的上一个/下一个关键字
      - 上记以外：原始功能，即如果页面有滚动条的话则上下滑动页面
